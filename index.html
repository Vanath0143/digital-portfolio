Demo front&back end web application

This project is a simple web app that uses:

  • ✅ HTML for structure(creating the form)
  • ✅ CSS for design(styling the form)
  • ✅ JavaScript for functionality(sending form data to the server)
  • ✅ Node.js with Express.js for the back end(handling form data and sending a response)
How it Works:
  1. The user fills out the form.
  2. When the form is submitted, JavaScript sends the data to the server.
  3. The Node.js server receives the data and responds with a message.
  4. The message is displayed back on the web page.
This setup is simple yet powerful for building interactive web applications.
